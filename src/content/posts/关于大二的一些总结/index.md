# 关于大二的一些总结

时间过得很快，距离我上次写总结，又过了一年（笑。五一闲来无事，来记录一下过去的一年吧。

## 6月

- 月初配置 Jittor 环境，花了好几天的时间。
- 分流结束，排名退了一些。
- 考了六级，寄了。然后就是复习期末考——记得高级语言程序设计 2-2，可能是输入输出流没背吧,拿了 3.7。

## 7月

- 期末考结束就是留在学校上了两周小学期（纯玩。
- 回到家被逼着去练车（最后也没学会。
- 之前写的 Qt 大作业可能是 vibe coding 的小试牛刀，到了这个月就是天天用了。
- 在 L 站看到了 Augment 的一个薅试用的方法，可以用 Claude 4 Sonnet，再加上一些神奇 prompt，以及一些奇妙 MCP，我在整个 7 月基本都在探索怎么使用 Claude 进行 coding。
- 然后在做新芽计划的代码复现部分，最开始选的是 RT-DETR，后面在换。
- 想到 8 月初能去参加 CCF 图形学夏令营还是激动不已的。

## 8月

- 月初去了深大，参加了一个夏令营，听了 3 天的讲座。
- 回来以后去考了科二，挂了。
- 然后把新芽计划复现的代码跑出来了，把汇报视频也录了。月中就联系了导师进组了。
- 我说想做生成，于是让我去和师兄做世界模型 / 视频生成。
- 师兄给了我一个 paper list，不过我没有好好补……

## 9月

- Augment 改为积分制，vibe coding 工具似了，转而使用 Cursor。
- 开学打了 3 天数模国赛，我选了我比较熟悉的物理题（折射相关），合作得不是很顺利，最后只拿了省一。
- 然后是评奖学金，出于班级的限制，40 × 0.1 = 4，差一名才能领果酱。

## 10–11月

- Cursor 太贵，转而使用 Copilot。
- 关于这两个月居然想不起来在干什么了。
- 也许是学业压力太大，每天都是满课，上课玩手机，下课放松休息。
- 没有找到自己需要什么！！！
- 还有离散数学！！每个周末都在花时间抄课件（坏习惯。
- 这段时间根本没好好看论文，都是周五下午要汇报了临时扫一下。

## 12–1月

- 使用 Antigravity，Google 家的 IDE。
- 重考六级，终于擦线过了。
- 开始处理了一些数据，让自己忙起来。
- 然后就是变态期末周，复习的人快玉玉了。
- 最后大语 3.0，好几门 3.7，排名又是一降再降。
- 月底有 Codex 了，我记得是 GPT 5.2。
- 期末考完那一两天用 Codex 速通了 BusTub，AI 太强大了（x。

## 2月

- 寒假挺闲的。突然对搜广推有兴趣，选了数据挖掘的课。
- 每天玩玩，然后跑一跑其他模型的推理。
- 月底 OpenClaw 突然火了，我试着接入了飞书，然后用 Codex 的 token，自动化地给一个开源项目提了几个 PR，不过没通过（乐。
- 哦还尝试了 geometry memory 引导生成的 idea。

## 3月

- 刚开学，想着这学期要过上线虫生活。
- 开始了游泳。
- 很想实习。

## 4月

- 清明节在压力之下，训练出了第一个 LoRA，不过第一次训练输入出错，第二次的不 work。
- 后面一个月就是玩玩玩了。中间电脑坏了，歇了一周（对学习没啥影响，乐。
- 这学期没有了做题课，99% 的作业都是 vibe coding。

## 5月

- 五一进行中，玩了三天。

## 展望

接下来还有半年的时间，希望能拼一把吧。唉生成方向太卷了，paper看不懂，code不会写。
每天打开小红书还有 peer pressure。
